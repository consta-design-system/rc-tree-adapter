# Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/rc-tree` или `npm install @consta/uikit @consta/rc-tree`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/rc-tree` были вложены в компонент `Theme`.


#### Всë!
