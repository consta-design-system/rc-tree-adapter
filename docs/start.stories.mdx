# rcTreeAdapter | Дизайн-система Consta

Адптер для дерева из библиотеки [rc-table](https://github.com/react-component/tree). Стилизует дерево для [дизайн-системы Consta](https://consta.design/).

[Посмотреть на GitHub](https://github.com/consta-design-system/rc-tree-adapter)

#### Как создать стилизованное дерево

Есть два варианта:

- Подключите миксин [RcTree](?path=/docs/rctree--page) — он добавит дереву стили из Consta.
- Подключите адаптер [rcTreeAdapter](?path=/docs/utils-rctreeadapter--playground), чтобы добавить стили и значения по умолчанию для некоторых свойств.

## Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/rc-tree-adapter` или `npm install @consta/uikit @consta/rc-tree-adapter`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/rc-tree-adapter` были вложены в компонент `Theme`.

[Как подключить тему](http://uikit.consta.design/?path=/docs/thematization-how-to-connect-theme--page)
