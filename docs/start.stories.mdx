# Компонент Tree для дизайн-системы Consta

Здесь можно посмотреть документацию и примеры компонента Tree — это дерево,
подходит для создания иерархических структур, например, каталога файлов, разложенных по папкам.

Компонент взят из библиотеки [rc-table](https://github.com/react-component/table) и адаптирован для [дизайн-системы Consta](https://consta.gazprom-neft.ru/).

[Посмотреть документацию и примеры](?path=/docs/components-tree--playground)

[Посмотреть на GitHub](https://github.com/gazprom-neft/tree)

## Быстрый старт

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/tree` или `npm install @consta/uikit @consta/tree`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/tree` были вложены в компонент `Theme`.

[Как подключить тему](https://consta-uikit.vercel.app/?path=/docs/thematization-how-to-connect-theme--page)

#### Всë!
