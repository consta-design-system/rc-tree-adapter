## Дерево

Создайте объект с описанием дерева:

- `key` — идентификатор ветки,
- `title` — название ветки,
- `children` — объект с дочерними элементами (если они есть).

Объекты вкладываются один в другой — так же, как будут вкладываться друг в друга ветки дерева.

```tsx
const data = [
  {
    key: '0-0',
    title: 'Ветка 1',
    children: [
      { key: '0-0-0', title: 'Ветка 1-1', children: [{ key: '0-0-0-0', title: 'Ветка 1-1-0' }] },
      {
        key: '0-0-1',
        title: 'Ветка 1-2',
        children: [
          { key: '0-0-1-0', title: 'Ветка 1-2-0', disableCheckbox: true },
          { key: '0-0-1-1', title: 'Ветка 1-2-1' },
          { key: '0-0-1-2', title: 'Ветка 1-2-2' },
          { key: '0-0-1-3', title: 'Ветка 1-2-3' },
          { key: '0-0-1-4', title: 'Ветка 1-2-4' },
          { key: '0-0-1-5', title: 'Ветка 1-2-5' },
          { key: '0-0-1-6', title: 'Ветка 1-2-6' },
          { key: '0-0-1-7', title: 'Ветка 1-2-7' },
          { key: '0-0-1-8', title: 'Ветка 1-2-8' },
          { key: '0-0-1-9', title: 'Ветка 1-2-9' },
        ],
      },
    ],
  },
]
```

Чтобы вывести дерево, передайте в свойстве `treeData` объект с данными.

Укажите идентификаторы узлов, которые хотите раскрыть, в `defaultExpandedKeys`. Если нужно раскрыть все узлы, добавьте `defaultExpandAll`.

Укажите префикс для CSS-классов в свойстве `prefixCls` (это обязательно). Если префикс не нужен, поставьте `prefixCls={''}`.

```tsx
export function UseRcTreeAdapterStoriesExample() {
  const treeProps = useRcTreeAdapter({ treeData: data, defaultExpandAll: true })

  return <RCTree {...treeProps} />
}
```

## Свойства

<RemadeTableInfo apiUrl="https://github.com/react-component/tree" />

| Свойство                          | Тип и значения | По умолчанию | Описание                            |
| --------------------------------- | -------------- | ------------ | ----------------------------------- |
| [`dataTree`](#дерево)             | `Data[]`       | -            | Объект с данными для дерева         |
| [`defaultExpandAll?`](#дерево)    | `bool`         | -            | Раскрывает все ветки                |
| [`defaultExpandedKeys?`](#дерево) | `object`       | -            | Раскрывает ветки по идентификаторам |
| [`prefixCls`](#дерево)            | `string`       | 'rc-tree'    | Префикс для CSS-классов             |

### Атрибуты веток

| Свойство              | Тип и значения | По умолчанию | Описание                       |
| --------------------- | -------------- | ------------ | ------------------------------ |
| [`key`](#дерево)      | `string`       | -            | Уникальный идентификатор ветки |
| [`title`](#дерево)    | `string`       | -            | Название ветки                 |
| [`children`](#дерево) | `object`       | -            | Объект с дочерними элементами  |
